# 智能健康分析系统

基于AI的在线健康报告分析工具，支持PDF上传和多种分析模式。

## ✨ 功能特点

- 📄 **PDF报告上传**：支持拖拽上传，自动提取文本
- 🔍 **四种分析模式**：TMT热成像、血液检测、综合报告解读、健康问答
- 🤖 **AI智能分析**：基于DeepSeek大模型，提供专业健康解读
- 💊 **产品智能推荐**：根据分析结果自动匹配适合的健康产品
- 🔒 **隐私保护**：纯前端运行，API密钥保存在本地

## 🚀 快速开始

### 在线使用
直接访问部署好的网站即可使用。

### 本地运行
1. 下载 `index.html` 文件
2. 用浏览器打开即可
3. 配置您的 DeepSeek API 密钥

## 🔑 获取API密钥

1. 访问 [DeepSeek开放平台](https://platform.deepseek.com)
2. 注册账号（支持国内手机号）
3. 在「API Keys」页面创建密钥
4. 新用户有免费额度

## 📊 分析类型说明

| 类型 | 说明 |
|------|------|
| TMT热成像 | 分析身体各部位温度分布，评估血液循环和代谢状态 |
| 血液检测 | 解读血常规、生化指标等，识别异常指标 |
| 综合报告解读 | 针对各类体检报告进行系统性解读 |
| 健康问答 | 针对具体健康问题提供专业建议 |

## 🛡️ 免责声明

本系统提供的分析结果仅供参考，不构成医疗诊断或治疗建议。如有健康问题，请咨询专业医疗人员。
